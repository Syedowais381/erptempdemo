# BizOS

BizOS is an enterprise-grade business management SaaS platform frontend built with React, Tailwind CSS v4, and Vite.

## Features
- **Dashboard**: Real-time revenue charts and alerts.
- **Inventory Management**: Track stock levels and add products.
- **Human Resources (HR)**: Manage employees, attendance, and payroll.
- **Supply Chain**: Manage purchase orders and outbound shipments.
- **CRM**: Track customer lifetime value and order histories.

## Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server.
